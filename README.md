# VoxAlic

一款轻量的《Warframe》世界状态监控桌面工具，全中文界面。

## 功能

- **世界时间** — 五大开放世界昼夜/状态周期
- **虚空裂缝** — 普通 / 钢铁之路 / 虚空风暴
- **赏金 & 回廊** — 各地点赏金奖励池、双衍无限回廊
- **仲裁 & 虚空商人** — 仲裁节点、Baro 货物与倒计时
- **订阅提醒** — 关注的内容出现时自动通知
- **后台常驻** — 托盘、开机自启、自动更新

## 安装

[Releases](https://github.com/SwingRainbow/VoxAlic/releases/latest) 下载 `VoxAlic_x.x.x_x64-setup.exe` 运行即可（已内置 WebView2）。
